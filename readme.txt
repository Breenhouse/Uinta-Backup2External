Updated 3/2/20

This script MUST be ran as admin from an external drive while
also logged into the target user's profile. Otherwise,
some grave errors may happen. So double check.

Close out of all applications, or some data may not
copy correctly.

Compatible with Windows 7 and newer.

This script will: 
* Create the default folder structure
* Copy background
* Copy Chrome app data
* Copy Firefox app data
* Copy Microsoft Signatures app data
* Google Earth Pro app data
* Copy Sticky Notes app data
* Copy user profile

Be sure to double check that you have all necessary data
backed up after using this script.
